# Mini CRM - Client Lead Management System

## 📌 Project Description
This is a Mini CRM application to manage client leads.
Users can add, update, and track lead status.

## 🚀 Features
- Add new leads
- Update lead status (New / Contacted / Converted)
- Notes & follow-ups
- Admin login 

## 🛠️ Tech Stack
- Frontend: React / HTML / CSS / JS
- Backend: Node.js / Express
- Database: MongoDB

## 📂 Project Structure
- frontend/
- backend/

## ▶️ How to Run
1. Install dependencies
2. Run backend: `npm start`
3. Run frontend: `npm start`
